# DealDish
DealDish is a full-stack web application designed to aggregate weekly supermarket prices, cross-reference them with a curated recipe database, and generate personalized meal plans. It helps users reduce grocery list-planning time and cut grocery spend.

## Tech Stack
- **Database:** MongoDB
- **Backend:** Node.js, Express
- **Frontend:** React
- **Features:** 
  - JWT Authentication
  - WebSocket notifications for real-time updates
  - Cost-optimized add-to-shopping-list workflows

## Project Structure
- `/frontend`: React application containing UI components, hooks, contexts, and API integration services.
- `/backend`: Node.js/Express server containing RESTful API routes, WebSocket integration, mongoose models, and business logic.
